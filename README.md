# Zorvyn Finance Dashboard
A modern finance dashboard to track income and expenses with visual insights.
## 🚀 Features
- 📊 Interactive charts (Line & Pie)
- ➕ Add transactions (Admin only)
- 🔍 Search & filter transactions
- 📈 Insights (highest spending, average, total)
- 👤 Role-based access (Admin / Viewer)
- 🎨 Responsive and modern UI
## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js
